# Historie (`history`)

- **Soubory:** history.html — Chart.js grafy, KPI #kpi-fills, empty-state
- **Počet stavů:** 31
- **biggestRisk:** Forecast watch + current-month souběh (history.html:560-562): pokud je hlídaný měsíc (forecastWatches) zároveň aktuálním měsíchem (currentPeriod), zlatý border (#fde047) přebije amber current-month outline. Tento stav nemá žádný mock-scenario ani smoke-test scénář, přičemž Notifier.processForecastWatches (notifications.js:172) zároveň spustí WS notifikaci — kombinace vizuální změny + notifikace má nulové testovací pokrytí. Pokud navíc latestForecast dorazí až po loadFillTypes() (race condition: REST call before first WS tick), _userPickedFt=false + beforeValue='' způsobí, že rebuildFillTypeDropdown() tiše změní selection i loadPriceChart() bez uživatelovy akce, a to potenciálně na jinou komoditu než tu, jejíž watch byl nastaven.
