# KPI lišta (`kpi`)

- **Soubory:** index.html (#kpi-balance, #kpi-time, #kpi-owned a další KPI karty — placeholder "—" vs hodnota)
- **Počet stavů:** 21
- **biggestRisk:** Hranice delta stavu — Zůstatek delta (podpora -0, 0, +0 hranic) není dobře zdokumentován; může nastat situace, kde farmBalanceDeltaDay je null/undefined vs 0 — logika na řádku 465-469 očekává Number.isFinite(delta), jinak text zůstane v innerHTML: ''. Tato hraniční případka není pokrytá scénářem mock-data. Navíc: flash-treshold (max(1000, 0.1 %) na řádku 594) není vůbec viditelný v UI — jen v kódu; uživatel neví, proč flash někdy nefunguje na malé změny. Třetí riziko: Předpověď počasí — CSS :empty selektor (472) skrývá #kpi-weather-forecast bez HTML, ale pokud server pošle prázdné forecast: [] místo null, element zůstane viditelný se selhání renderů. Čtvrté: Měna symbol() — pokud FS25Money není naloadován v čase renderKPI(), fallback je '€', ale ostatní části kódu mohou FS25Money expect — data misalignment.

> Povrch: KPI lišta (Hero row): 4 karty s Live daty — Zůstatek, Počasí, Den, Čas
>
> **Poznámka:** `#kpi-day-sub` spojuje části oddělovačem `' · '` (mezera-tečka-mezera), ne prostou mezerou. Reálný výstup: `'březen · jaro · 1. rok'`.
