# Zvonek (`bell`)

- **Soubory:** bell.js (CELÝ — prahy FOOD_LOW=25/WATER_LOW=20/STRAW_LOW=25/OUTPUT_HI=90/FUEL_LOW=15/STORAGE_HI=95, genAlerts, severity urgent/warning/info, fingerprint+dismiss, badge #bell-count, hidden vehicles přes TableTools.getHidden)
- **Počet stavů:** 21
- **biggestRisk:** Vehicle low-fuel alert (veh-fuel, urgent) is gated by TableTools.getHidden('vehicles') / DashState hiddenVehicles AND keyed on vehicle NAME (String(v.name)), not a stable id. Two distinct risks compound here, neither covered by any spec beyond the single low-fuel.spec.js happy path: (1) the hide-set is matched by name, so two vehicles sharing a name (e.g. two identical Fendt 942 Vario) collide — hiding one hides BOTH from the bell, and the dismiss fingerprint (key::sorted-names) also merges them; (2) the gate has a silent fallback chain (TableTools → DashState → empty Set) meaning on any page where neither TableTools nor DashState is loaded, a hidden vehicle still raises the urgent alert. This is the only cross-surface gated branch and the only urgent severity, so a regression here either spuriously fires the highest-priority alert or silently swallows a real low-fuel warning. Additionally the seven info/warning alerts (anim-food/water/straw/output, field-ready, storage-full) and all boundary/dismiss/empty states have ZERO spec coverage — only the fuel path is tested.

> Povrch: Zvonek (notification bell) — bell.js renders a nav button (#bell-btn) with a count badge (#bell-count) plus a drop-down panel (#bell-panel) whose list (#bell-list) holds one <li.bell-item> per alert group. genAlerts() produces 7 possible alert groups across 3 severities (urgent/warning/info). All thresholds are exclusive < (low) / inclusive >= (high). Coverage: state/bell.spec.js drives every threshold boundary (low/ok) via the dedicated st-bell-* scenarios, all 3 severities' icon recolor, the hidden-vehicle gate, individual + dismiss-all, and the row-click card-flash. Remaining gaps: 9+ badge overflow (unreachable — only 7 alert keys), cross-page deep-link redirect from side pages, and the dup-name hidden-set collision (biggestRisk).
