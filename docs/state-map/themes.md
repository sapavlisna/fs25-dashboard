# Témata (`themes`)

- **Soubory:** theme.js + style.css (5 témat dark-green/dark-blue/light/high-contrast/fs25-native, atribut data-theme, tokeny --bg/--surface/--accent)
- **Počet stavů:** 81
- **biggestRisk:** Flash-down větev pro vehicles (vozidlo odevzdalo náklad / tankuje → implement se vyprazdňuje) — je implementována přes noteImplementFlashes() + getImplFlashCls(), ale žádný scénář v mock-scenarios.js ho nespustí v sestupném směru; wagon-filling jede jen 0→100 %, nikdy 100→0 %. Tím pádem flash-down na vehicle-row.flash-down není nikdy ověřen automatizovaně. Navíc getImplFlashCls() bere last-wins z Map iteration order (komentář v kódu to přiznává) — pokud dvě fillUnits jednoho vozidla flashují v opačných směrech zároveň, výsledná třída je nedeterministická a závisí na pořadí iterace Map.
